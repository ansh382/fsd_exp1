# FSD-I Experiment 1.3

## Title
Admin Dashboard using CSS Grid and Theme Switching

## Description
This experiment focuses on designing a simple admin dashboard using CSS Grid.
A light and dark theme switching feature is implemented using JavaScript to
change the appearance dynamically.

## Learning Outcomes
After completing this experiment, I learned:
- How to create page layouts using CSS Grid
- How to design an admin dashboard interface
- How to apply light and dark themes using CSS
- How to use JavaScript for dynamic theme switching
- How to organize frontend code properly
- How to use GitHub to upload project files
- How to deploy a web project using Netlify

## Technologies Used
- HTML
- CSS
- JavaScript
